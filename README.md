# 🩸 Nakla Sherpur Blood Bank

![Nakla Sherpur Blood Bank](https://dev-mahir.vercel.app/assets/nsbb.png)

This platform was built to help people in Nakla-Sherpur and surrounding areas quickly connect with suitable blood donors during emergencies. Select your required blood group to find nearby donors. One-click direct calling is available. You can also register as a blood donor.

### 🌐 Live Website

https://nakla-sherpur-blood-bank.vercel.app/

### ⚡ Tech Stack

Html • CSS • Tailwind CSS • Javascript • Firebase
